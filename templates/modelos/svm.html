Support Vector Machine (SVM)

Support Vector Machine (SVM) es un modelo de clasificación supervisado basado en la teoría estadística del aprendizaje.

Busca encontrar el hiperplano óptimo que maximiza el margen entre clases y puede usar kernels para clasificaciones no lineales.

Es ampliamente usado en procesamiento de imágenes, voz, series temporales, minería de datos y más.
